Git

  • git init - переводится как "Инициализация нового репозитория". Создает новый репозиторий Git в текущей директории.
  • git clone ссылка с GitHub - переводится как "Клонирование удаленного репозитория". Копирует существующий репозиторий на локальный компьютер.
  • git add "название файла" - переводится как "Добавление изменений в индекс". Подготавливает изменения для следующего коммита.
  • git commit -m "Сообщение коммита" - переводится как "Создание коммита". Сохраняет изменения в истории репозитория.
  • git status - переводится как "Проверка состояния репозитория. Показывает текущие изменения, которые еще не закоммичены.
